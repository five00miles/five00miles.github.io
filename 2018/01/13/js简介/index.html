javascript简介

与java没有任何关系，当时java很火，为了搭顺风车，就加个java

JavaScript和ECMAScript的关系：ECMAScript是一种由Ecma国际前身为欧洲计算机制造商协会，英文名称是European Computer Manufacturers Association,制定的标准。Javascript是由公司开发而成的，公司开发的一定又一些问题，比如不便于其他公司扩展和使用，所以欧洲的这个ECMA的组织，牵头制定JavaScript的标准，取名为ECMAScript。简单来说ECMAScript不是一门语言，而是一个标准。符合这个标准的常见语言有：JavaScript、Action Script(Flash)。

ECMAScript再2015年6月发布了ECMAScript6版本，即ECMAScript2015

js是一种专为与网页交互设计的脚本语言，由三部分组成：

  • ECMAScript -提供核心语言功能
    • 语法
    • 类型
    • 语句
    • 关键字
    • 保留字
    • 操作符
    • 对象
  • 文档对象模型（DOM） -提供访问和操作网页内容的方法和接口
  • 游览器对象模型（BOM） -提供与游览器交互的方法和接口
    其编写的程序可以直接在游览其中解释执行

js文件写在<script></script>标签里面，这个标签放在哪里都行，一般放在<head></head>标签里面或者放在</body>前面

<script></script>

1.script 可选属性：

  • type
    • 值：MIME-type -指示脚本的MIME类型
    • 例子：<script type="text/javascript"></script>
  • async(HTML5)
    • 值：async -规定异步执行脚本（仅适用于外部脚本）
    • 例子：<script async="async"></script>-本相对于页面的其余部分异步地执行（当页面继续进行解析时，脚本将被执行）
  • charset
    • charset -规定在外部脚本文件中使用的字符编码
    • 例子：<script charset=utf-8></script>
  • defer
    • defer -规定是否对脚本执行进行延迟，直到页面加载为止
    • 例子：<script defer="defer" src="https://cdn.bootcss.com/jquery/3.2.1/jquery.js"></script> -加载的jquery延迟执行，等整个页面加载完成之后，这个js再执行
  • src
    • URL -规定外部脚本文件的URL
    • 例子：见上，就是加载外部js文件
  • xml:space
    • preserve -规定是否保留代码中的空白
    • 例子：<script xml:space="preserve">
    • 不常用我不会

如果既不使用 async 也不使用 defer：在浏览器继续解析页面之前，立即读取并执行脚本

2.<script>的一些问题

  • 再代码中直接使用</script>

  • <noscript></noscript>:只在1.游览器不支持脚本。2.游览器支持脚本，但脚本被禁用时，noscript标签内的内容才显示

  • 在XHTML中使用js
    • 在XHTML中，<会被解析成开始一个新标签，所以可以使用&lt来代替,其他同理
    • 另一种是使用CData片段来包含js代码
