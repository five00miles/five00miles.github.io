js的数据类型

发表于 2018-01-15 | 分类于 javascript

js变量

  • js是弱类型的脚本语言
  • 变量的命名规则
    • 以字母或者下表符号_或者$为开头
    • 变量名长度不能超过255个字符
    • 不允许使用空格
    • 不使用脚本语言中保留的关键字和保留符号作为变量名
    • 变量名区分大小写
  • 全局变量
    • 在方法外部声明的变量
    • 方法内部，没有加var关键词声明的变量
  • 局部变量
    • 方法内部，使用var声明的变量

数据类型（6种）

1.基本数据类型

  1. undefined：值未定义
    • 变量声明了，但是没有被赋值
  2. null：空值，也是一个空对象
  3. boolen：布尔值

  4. number：数值

    • 整数，小数（最高精度17位小数），NaN，Infinity,-Infinity
    • var a = 070：表示八进制；var a = 0xA：表示十六进制
    • 不要做if(a+b==0.3)这样的判断，尽量避免小数判断

  5. string：字符串

2.引用类型

  1. Object：比如对象、数组、Date、正则表达式

3.数据转换

  • ==：两个等号表示可以进过自动转换，比较最后转换成的数值
  • ===：三个等号表示可以进过自动转换，先比较最后转换成的数值，再比较数据类型

涉及的函数

  • typeof：判断数据类型
  • parseInt：将字符串转化成数据类型
